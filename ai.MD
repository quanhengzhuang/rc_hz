AI 使用记录

## 确认设计思路
将需求分别提交给 deepseek 和 gemini，对比和自己的设计思路，查漏补缺。

工具： deepseek 和 gemini
prompt:
```
请理解以下需求，说一下实现思路，未来将会用 Go 语言实现。

// README.md 中的「需求」部分
```

## 确认详细设计
将 README.md 中的「详细设计」部分提交给 AI，参考其代码实现，完善自己的详细设计。

工具： deepseek 和 gemini
prompt:
```
以下是我最新的设计，请给出实现代码，代码要可运行。

// README.md 中的「详细设计」部分
```

## 给出代码实现
将 README.md 中的「详细设计」部分提交给 AI，给出具体的代码实现，并提交第一版代码。

工具： TRAE
prompt:
```
以下是我最新的设计，请给出实现代码，代码要可运行。

// README.md 中的「详细设计」部分
```
